# Участие в проекте

Спасибо за интерес к МастерБот! Мы рады любому вкладу.

## Как начать

1. Форкните репозиторий
2. Создайте ветку: `git checkout -b feature/your-feature`
3. Поднимите окружение: `make up && make migrate && make seed`
4. Внесите изменения
5. Запустите тесты: `make test`
6. Запустите линтер: `make lint`
7. Создайте Pull Request

## Правила

- Код на Python 3.12+, типизация обязательна
- Форматирование: ruff
- Тесты: pytest, покрытие критической бизнес-логики
- Миграции: Alembic, одна миграция = одно изменение
- Коммиты: осмысленные сообщения на русском или английском
- PR: описание что и зачем, связь с issue если есть

## Модульность

При добавлении нового модуля:
- Создать директорию в `app/services/` или `app/models/`
- Зарегистрировать в module registry
- Добавить feature flag
- Добавить тесты
- Обновить документацию

## Вопросы

Открывайте issue для обсуждения перед крупными изменениями.
