# Права и роли

## Роли

| Роль | Описание | Мультироль |
|------|----------|:---:|
| `product_owner` | Полный доступ к платформе | + admin |
| `admin` | Управление каталогом, пользователями, настройками | + master |
| `senior_master` | Управление веткой мастеров, согласование скидок | + master |
| `master` | Создание смет, поиск по каталогу | - |
| `client` | Просмотр каталога, создание заказов | + любая |

## Правила иерархии

1. `senior_master` назначается только `admin`
2. `senior_master` видит только свою ветку
3. `master` принадлежит одной ветке или напрямую `admin`
4. `admin` может переводить мастера между ветками
5. Увольнение через `senior_master` требует подтверждения `admin`
6. `product_owner` имеет полный доступ ко всему
